## Mandarin Translation 中文翻譯



### 第一段：\[00:00:00] -

Gui Prada： 但接著我們會開始鑽牛角尖，想著：「喔，我的舞伴會怎麼想？」或是「等一下，現場的人會怎麼看我？我會不會惹上麻煩？」有一種想法是，你和舞伴進入了一個「泡泡」，其他都不重要了——誰在看、別人在想什麼，我都不在乎。我也曾把自己放進那個「專業框架」裡，列出一清二楚的清單：這些是我被允許做的，那些是不被允許做的。 想像一下，我們正在跳舞，舞伴就在我懷裡哭泣。現在請想像，當我說出有人在跳舞時在我懷裡哭泣，你的大腦會產生多少種負面意義？ 我認為這就是核心秘密：你是否有能力「觀察」身體、觀察當下的狀況，而不去給它「貼上意義」。我也曾遇到在節慶活動現場人山人海的情況，我也會想：「喔，他們會怎麼想？他們會覺得我想跟這個人約會，然後明天我就會在社交媒體上被公審（cancelled）。」 問題在於，我們為發生的事情附加了意義，這就是一切混亂的開端。在我看來，人生被兩種強大的力量驅動：我們竭盡全力避免「看起來很糟」，以及努力嘗試「看起來很好」。

Alisson Sandi： 你如何在舞池中做自己，而不必擔心誰在看？你能在與舞伴待在泡泡裡的同時，依然尊重房間裡的其他人嗎？你該如何停止過度思考，單純享受深層的連結？讓我們一起來探討。 大家好，歡迎回到 _ZoukNerds_。在本集節目中，Gui 和我將討論在尊重界限的同時尋找自由。我們會討論 Gui 如何保持在當下（stay in the moment），以及我自己在舞感特別強烈時所感受到的被評判（feeling judged）的掙扎。讓我們來探索如何傾聽舞伴的身體，享受連結且不再感到壓力。我們開始吧！

（片頭音樂）

Alisson Sandi： 大家好，我是 Alisson Sandi，讓我們歡迎美麗的搭檔 Gui。嗨 Gui。

Gui Prada： 嗨 Alisson。大家好，歡迎回到 _ZoukNerds_。

Alisson Sandi： 在本集節目中，我們要討論一件在我們社群中非常、非常普遍的事情。最近我們與學生、舞者、社群的朋友們討論過，那就是有時候當我們在跳舞時，音樂、環境、舞伴以一種非常特別的方式啟發我們，讓我們想要以某種方式表達——可能是能量、流動或是情緒。 但接著我們會想太多：「喔，我的舞伴會怎麼想？這可以嗎？這會不會變成問題？」或是「等等，現場那些看著我們跳舞的人會怎麼想？這被允許嗎？我會不會惹麻煩？」清單沒完沒了。 所以，我們如何在與舞伴及周遭人事物達成共識、相互尊重的前提下，自由地表達自己？如何感到自信並與舞伴交流？ 我們該怎麼做？因為我知道一件事，我們以前也討論過，你（Gui）有一種「佛系」技能，能夠將你和舞伴與房間裡的其他部分隔離開來，你們兩個有時能進入另一個宇宙。那是怎麼運作的？幫我們這些還沒有這種超能力的人分析一下。

Gui Prada： 喔，我認為每個人都有這個能力，只是還沒完全與這種能力連結。這包含兩件事：第一是進入泡泡，不在乎誰在看或想什麼。但並非完全「不在乎」，而是要考慮環境。 舉例來說，如果在一個只有 15 個人的小社群、小舞池，你和同一個人跳了一個小時，那就會產生影響，因為能跳舞的人就那麼多，在這種情況下，你需要多留意周遭的人，尤其是你的舞伴。 幫助我最大的主因是「不去對某些發生的事情附加意義」。例如，這曾發生過幾次：我正在跟某人跳舞，我察覺到對方的呼吸改變了，接著我發現對方開始哭了。我們還在跳舞，那個人就在我懷裡哭。 就像我剛才說的，光是聽到這件事，你的腦海裡可能已經對「跳舞時有人在哭」這件事產生了無數種解讀。在我的腦袋或身體裡，發生的事是這樣的：「喔，這個人在哭，就這樣。」我不會去想「喔，她哭是因為難過」或「她哭是因為發生了什麼事」，我只會問：「一切還好嗎？」對方說：「嗯，還好。」對話就結束在那裡。這就是核心技能。

Gui Prada： 所以，如果我在跳舞時，感覺到舞伴身體緊繃，我的大腦會反應：「喔，身體緊繃了。」而不是去想：「喔，她在拒絕我」或「我做了什麼不好的事」。這不是一種審判（judgment），而是觀察發生的事並適應它。 我追求的是身體上的自在（ease on the body）。透過隨時留意舞伴的反應，我能調整流動，而不附加意義。當然我不總是成功，我也不是什麼開悟大師，但大多數時候我能保持觀察。反之亦然，如果我們從遠距離開始，隨時間越來越靠近、越貼近，我會專注在物理感覺與接觸點的觀察，感受到那種美好，但不會立刻去想：「喔，我們對彼此有意思」或「這代表了什麼」。 我單純留在那個體驗中。因為這麼多年後，我理解我們處於一個「容器（container）」中，這個容器允許我們達到平常無法達到的親密與靠近。在這個容器裡，我們被允許聽從身體，而音樂、環境、以及最重要的「技術工具」給了我們安全感與意識。

### 第二段：\[00:10:00] -

Gui Prada： 最後我想分享的是，理解我們所做的這項活動——雙人舞，特別是 Brazilian Zouk、Forró 等包含擁抱的舞蹈——它的目的是滋養我們的身體。我們是需要身體接觸才能存在的生命，而雙人舞提供了一個安全、溫柔且滋養的環境。 我記得以前我還在當律師時，我每週唯一的社交舞就是週五與一位女性友人跳 45 分鐘。我們會約在門口見面，跳整晚，有時計時 45 分鐘，然後我就回家。因為我整週都坐在辦公桌前對著電腦、數字、法律案件，握了無數次禮貌而疏遠的手。 我唯一的身體滋養就是回家擁抱我媽和兄弟，但我需要更多。我需要結合動作、創意與表達的身體連結，且我希望它是安全、可靠且有趣的。對我來說，那是我釋放一整週壓力的管道。 週五晚上跳完舞回到家，我會吃披薩，看《Globo Repórter》（巴西週五晚間的紀錄片節目）。對那時的我來說，舞蹈就是滋養。 所以，秘訣（也是最難的部分）是觀察身體與當下，不附帶意義。如果你不習慣與親密舞伴、家人或朋友以外的人有這種層級的連結與溫柔，這確實很難。當你跟一個陌生人跳舞，感受到那種連結，你可能會想：「天哪，我好喜歡這個人，我想抱緊一點。」但接著會想：「喔，但這個陌生人會怎麼解讀我的擁抱？」這是非常合理的擔憂。

Gui Prada： 學習「留在觀察中（staying in the observation）」是第一步。下次當你在社交舞池觀察別人跳舞時，看到一個人的手放在另一個人的脖子上，你會覺得「喔，他們一定在交往」？還是你只能看到「那是一隻放在脖子上的手」？

Alisson Sandi： 這真的很難。在錄音前我也跟你分享過，我的大腦永遠在過度思考生活中的一切。我有時在舞池裡感受到某種情感，想要以某種方式連結，但我腦中會立刻出現：「不行，這不被允許」或「身為一名舞蹈老師，我不該這樣做」。 當我周圍都是學生，或是身處在一個大型節慶（Festival）現場，我會想：「喔，他們會怎麼看？他們會覺得我想跟這個人約會，然後明天我就會在社交媒體上被公審。」 作為一名藝術家，我會想這些；但身為學生或一般舞者，我們也常聽到這種擔憂，擔心對方感受，更擔心別人怎麼看自己。 這很合理，也是正面的，因為我們希望有清晰的界限、相互尊重的舞蹈，以及安全的社群。但如何表達自己？如何達到那種雙方都感到足夠安全的境界？ 第二個問題是相反的：有些人對這種親密「太過自在（too comfortable）」。這反而在社群中變成問題。有些人會說：「嘿，我來這是為了探索，為什麼我要被你們說的『社群安全』限制？」我們甚至看過有人被禁止參加社群、課程或活動，因為他們過度跨越界限。 Gui，你如何定義那個光譜？從這裡到這裡是可以的，從那裡開始你需要確認，而那裡則是禁區？

Gui Prada： 你指的是非言語（non-verbal）的方式嗎？

Alisson Sandi： 讓我們開放討論，可以從非言語開始，再到言語溝通。

Gui Prada： 這很難，但我從「理想情境」開始說起。理想世界中，在雙人舞場景裡，我們應該對身體不自覺發出的「舒服」或「不舒服」訊號非常敏感。 當我提到「不給張力附加意義」，我的意思是「傾聽身體」。當張力產生了，我就做出回應。如果舞伴用張力來拉開距離，我就停在那個距離，不去嘗試再次靠近。 重點在於，當我留在那個距離時，我的腦袋不會想：「該死，我們現在分開了，我被拒絕了」或是「我們應該要靠近才對，她不懂怎麼跳，讓我教她怎麼跳」。如果你這樣想，你就是在強迫。 在理想情況下，我們先聽從身體訊號並做出回應，不去編造「為什麼身體會這樣反應」的故事。反之，當對方感到很舒服、很放鬆時，有些人會想：「喔，這裡有個空間，讓我做些什麼吧，因為她肯定對我有意思。」

### 第三段：\[00:20:00] -

Alisson Sandi： 這就是問題所在，有些人會說：「這是連結的舞蹈（connection dance），為什麼要有極限？」

Gui Prada： 那是另一回事。那是帶著預設（should）而來的人。我說的是，隨著舞蹈進行，慢慢變得放鬆、投降（surrender）於連結中，比如臉貼臉或頭靠在胸前。 當有人帶著「我們必須靠近，因為這就是這種舞的樣子」的想法來跳舞，他們會停止傾聽身體，跳過建立關係的步驟。 我的老師（我想你的老師 Xandy Liberato 以前也說過）常說：每一支舞都是一段在歌曲進行中完整發展的關係。當有人帶著預設而來，他們像是直接跳到了關係的中段或末段，不給彼此時間去發展。 在理想情況下，大家應該留意那種覺知（mindful presence）。你可能不知道舞伴是否真的不舒服，但你能感知到自己的身體是否在強迫某些事。強迫靠近或強迫拉開距離，感覺都是不對的，這時你的身體正在告訴你一些訊息。

Gui Prada： 第二件事是理解你自己的極限（boundaries），這非常、非常具有挑戰性。我們受到「避免看起來很糟」和「努力看起來很好」的力量驅動，有時這會凌駕於我們身體真正的需求。 最近我一直在調適我內在的力量，去理解「我想做的」與「我願意做的」之間的差別。有時你不想做某事，但你「願意」嘗試一下看看感覺如何，或者意識到「我不想這樣」。這需要溝通——包含身體的與口頭的（例如簡單地說「不用了，謝謝」）。 了解是什麼在邀請你留下，是什麼在邀請你離開。如果離開的衝動被「喔，這個人是高級舞者，我應該繼續跳」或「有人在拍影片，我應該繼續跳」等念頭蓋過，那就是大腦在凌駕身體的直覺。當你離開那個情境後，不要責怪自己沒守住界限，而是去思考為什麼當時守住界限這麼難。

Alisson Sandi： 這很有深度。我想到另一個場景：當我是領舞者（Leader）時，我對於連結的深度非常敏感且保守。如果有從舞者（Follower）一開始就尋求極度深層、火熱或親密的連結，我會感到驚訝，並立刻縮回我的「安全區」。那種感覺有點嚇人，甚至我會變得非常守舊（conservative）。 但我意識到，當我是從舞者時，我變得非常開放。只要不讓我的身體受傷，我想體驗任何事——親密的、火熱的、玩樂的。 我在想為什麼會這樣？第一點是：我的主要角色是領舞者，這是我在社群中的形象，所以我希望在那個角色下被看成是專業、得體的。而從舞者是我的第二角色，即使我跳得不好或做了一些傻事，那不是我的主要標籤。 第二點是：作為領舞者，我常被期待提供體驗；而作為從舞者，我想卸下那些責任，單純去體驗對方帶來的東西。我會先看這場舞蹈是什麼「菜系」，然後再做出調整。

Gui Prada： 這很有道理。作為從舞者，故事和框架都變了，那是一個「零壓力」的角色。在雙人舞的許可範圍內，你給了自己更高的權限去享受。領舞者有責任，大家都在看領舞者，但如果是 Gui 在跳從舞，大家只會笑著說：「喔天啊，Gui 在跳從舞耶！」 這與我的經驗產生了強烈的共鳴。當我跳從舞時，大腦的開關變了，壓力小很多。當我遇到信任且喜歡的領舞者時，我甚至能比當領舞者時更靠近對方。我會運用所有的工具，且恐懼更少。

### 第四段：\[00:30:00] -

Gui Prada： 關於從舞者的視角，我還看過兩種類型的發展：一種是隨著時間慢慢建立信任，從遠距離開始，漸漸看到手臂變柔軟、頭靠得更近、眼睛閉上，那是一種緩慢的建立過程：「你在照顧我的身體，這裡，繼續照顧它。」 另一種是從舞者一開始就說：「好，我信任你，帶我去任何地方，直到你做了一些破壞信任的事為止。」 但我最近觀察到第三種：為了避免誤解或麻煩，全程保持距離。即使他們內心渴望信任、渴望靠近，但出於保護層，他們選擇留在那個安全的狀態。這讓我感到難過，因為 Brazilian Zouk 是如此美麗，它提供了一個尊重且溫柔的環境。

Alisson Sandi： 你剛才說的這點真的讓我很有感觸。有些時候，我也很想繼續深入那段連結，但我會主動喊停，不是因為我不想要，而是因為我覺得身為社群成員的壓力。 我和舞伴處於那個泡泡裡，感覺其他都不重要，但我的腦袋會跳出來說：「不，現在不對，這裡不對，現場有很多人，我們是在社交舞會（social），我們應該尊重環境，不要讓別人感到不舒服。」 或者我會擔心：如果我們表現出這麼深層的連結，當這些人跟我跳舞時沒感受到同等的連結，他們會不會覺得自己跳得不夠好？ 我的腦袋裡有太多這種事了。在我的職業生涯前十年，我創造了一個盒子把自己關起來，列出清單。過去十年，我一直試著在這個盒子外找到舒適點。 作為社群的藝術家，我們聽到了很多八卦和回饋，我會立刻把它們放進我的清單：「絕對不要說這個、絕對不要做那個」。我在尋找解決方法，而我的靈光一閃告訴我——這一切都與「情境（Context）」有關。 如果你是在私人課，學生想練習連結，且你們都同意誠實溝通，那就是合適的情境。或者在工作室裡練習，沒有相機、沒有旁人，那是安全的容器。 但在社交場合、節慶，如果你身為藝術家，只跟某個人有極其特別的體驗，而跟其他人沒有，我會擔心讓現場的人感到不平衡。

Gui Prada： Alisson，我們之後真的需要好好聊聊，這段故事聽起來很沉重（harsh）。我們剩下的時間不多，但我能說的是，我們有太多的「萬一」。這些萬一多半源於「努力看起來很好」和「避免看起來很糟」。

Gui Prada： 我記得我上次去台北（Taipei）時，在那個教 Hustle 和 Swing 的場地（注：可能指台北的某些複合式舞蹈空間），最後幾首歌我跟 Ting（注：Alisson Sandi 的妻子，也是知名舞者）跳舞。我當時像是「被附身（possessed）」一樣，那種感覺太棒了，簡直是進入了完美的流動狀態（state of flow）。 我記得有人在看我們，但我告訴自己的故事是：我正在享受最大的樂趣，而那些觀看的人是在尋找靈感。 在你剛才的故事中，你覺得他們會不開心，因為他們無法與你達到同樣的境界。但另一種說法是：他們看到我們在享受這種狀態，這或許能啟發他們也去與別人尋求這樣的連結。 我了解那種「我也想要有 Gui 的跳舞體驗」的期望。但我漸漸明白，要達到那個境界，需要很多因素配合。那次台北的社交舞會氣氛很棒，大家都在那裡。

### 第五段：\[00:40:00] -

Gui Prada： 看到他人擁有神奇的體驗，能成為一種啟發。這幫助我即使在有人注視的情況下也能享受其中。初學者看到有人開心地大笑、瘋狂地跳舞、親密地靠近，他們會想：「那是誰？我也想要那樣。」 雖然當他們來跟我跳舞時，我不一定能給他們完全一樣的體驗，但我可以給他們「一部分」那樣的能量。因為我的「電池」已經被剛才那場舞充飽了，我反而更能提供好的體驗給接下來的舞伴。 就像你跟 Ting 跳舞一樣，那種滋養會讓你能量滿溢，當你看向下一位舞伴時，你會覺得：「來吧，準備好接收我的能量！」

Alisson Sandi： 我可以理解那些火熱、激情的舞蹈。但一旦進入感性、脆弱、緩慢且親密的舞蹈時，我就會變得非常自覺、感到愧疚。 如果是跟我太太跳火熱的舞，我會覺得：「來吧，大家看，這很好玩！」但如果是跟她，或是跟任何人進入那個深沉、擁抱、緩慢的境界，那種罪惡感就會跑出來：「這不是合適的場所，我不該讓別人產生某些負面聯想。」

Gui Prada： 在這種情況下，因為「情境」是那是你太太，大家會覺得「喔，那是他老婆，難怪」。但如果是跟其他人，我能理解那種感覺。 這種啟發也可能是：「哇，你看他們多靠近，我也想要那樣。」 我們可能需要另一集節目來討論這個。特別是關於靠近（closeness）、緩慢（slowness）的層次。這不一定是指動作慢，而是建立連結的節奏。 我們沒時間了，也沒有罪惡感（no guilt）了！

Alisson Sandi： 那我們把這部分留給下一集。各位，謝謝你們的收聽。如果你喜歡我們的 Podcast，請幫我們一個忙：追蹤、訂閱、分享給你的朋友。這對於社交媒體的演算法很有幫助。 我們希望每週繼續做下去，因為這對我們來說很有趣，你們也告訴我們這對你們有幫助。 謝謝大家，親親（Beijos），拜拜！

Gui Prada： 再見！
