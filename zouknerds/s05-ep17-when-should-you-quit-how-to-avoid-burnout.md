### 第一部分：\[[00:00](http://www.youtube.com/watch?v=FgKjYCFWX8g\&t=0)] - \[[10:00](http://www.youtube.com/watch?v=FgKjYCFWX8g\&t=600)]

Alisson Sandi: 你要如何在對某件事保有熱情的同時避免職業倦怠？又或者，到了什麼程度你會選擇放棄？讓我們一起來探討。 歡迎回到 Zouk Nerds。在今天的節目中，我們將討論如何終身追求一份熱情，並達到你渴望的水平且避免倦怠。同時，我們也要關注那些引導你走向臨界點、有時甚至讓你完全放棄的原因。讓我們開始吧。

哈囉大家好，歡迎回到新的一集。我是 Alisson Sandi，今天我們有特別的共同主持人 Gui Prada。

Gui Prada: 哈囉大家，歡迎回來。

Alisson Sandi: 今天我們要延續上一集的話題。如果你聽過上一集，我們討論了讓你想要訓練、進步並在技能上投入更多精力的催化劑或理由，無論那是社交舞、教學或生活中的任何事。 但今天我們想把硬幣翻轉過來，或許會談得嚴肅一點。我們想討論是什麼讓你不想再做了？不想再投入時間、金錢和生命？以及是什麼會讓你徹底放棄？關於這點，我不僅有個人經驗，也有很多觀察他人的故事可以分享。Gui，這個主題對你有什麼啟發嗎？

Gui Prada: 這確實是一個有趣的話題。我最近常和親近的朋友聊天，他們分享了為什麼考慮停止參加活動、停止社交舞，或者在教學和測試上休息一段時間。所以這個對話來得很及時。 我想我們可以先從輕鬆一點的角度開始。以我的經驗來說，停止投入大量時間的原因——延續我們上一集聊到的——是當我對現狀感到滿足時，那種「必須不斷變強」的焦慮感就減輕了。我開始上課是因為我享受上課的體驗，喜歡那些相處十年的老師和同學。比起實際的訓練，我更喜歡那群人。當我對自己擁有的知識感到滿足，我就覺得 OK 了。

但這通常會有第二階段，就是大腦會跳出來說：「不，永遠不夠，你應該學更多。」然後你就覺得：「喔該死，我得去訓練了。」 對我來說，那時下班後非常疲累，大腦想追求進步的那部分根本無法啟動。唯一說話的聲音是：「我們能不能跳一下舞來放鬆，然後回家睡覺？」那個聲音太強大了。

Alisson Sandi: 關於這點，我要拿杯咖啡來細說了。延續我上一集提到的故事，我職業生涯中有一段時間在澳洲，那時我訓練的動力完全是處於「生存模式」。 當時我有很好的職業機會，但我沒有合法的長期居留權，因為我拿的是學生簽證。我想，如果我能成為全國甚至全球最強的舞者，我就能拿到簽證，在澳洲生活。那是支撐我很久的動力。 後來我和搭檔真的拿到了簽證，那天我們都哭了，覺得這會改變我們的一生。但接下來呢？我們的目標消失了，因為我們跳舞的初衷與情感是斷聯的。 當時澳洲社群的一位教練建議我們去挑戰世界錦標賽。我患有過動症（ADHD），一聽到挑戰就覺得多巴胺湧現，這成了下一個目標。但這依然與我的個人情感目標脫節。 在贏得四次世界冠軍後，我們感到前所未有的空虛。最終我和搭檔拆夥了，因為我們的個人生活、舞蹈與情感完全失衡。

在 2015 到 2017 年間，我徹底放棄了舞蹈。我沒辦法社交舞、沒辦法教學，我找了一份辦公室行政工作。在那一年多的時間裡，我開始審視內心，做了很多心理諮商，讀了很多書。 這讓我重新找回了那個年輕的舞者，那個因為「與人連結」而愛上舞蹈的自己，而不僅僅像一台工作的機器。

### 第二部分：\[[10:00](http://www.youtube.com/watch?v=FgKjYCFWX8g\&t=600)] - \[[20:00](http://www.youtube.com/watch?v=FgKjYCFWX8g\&t=1200)]

Alisson Sandi: 現在回到整個大社群。常有朋友來找我說：「我感受不到熱情了，不想去上課或社交舞，我哪裡做錯了？」 當他們處於這種黑暗深淵時，我問的第一件事是：「你能休息一下嗎？」 計劃在四週或兩個月後徹底抽離這個自動導航的環境。抽離後觀察發生了什麼：你懷念什麼？不懷念什麼？你真正重視的是什麼？你當初開始的初衷，和現在支撐你的理由還是一樣的嗎？這種自我研究對我和我的同儕非常有幫助。 我先停在這裡，讓 Gui 分享你的經驗，以及你對社群朋友的觀察。

Gui Prada: 聽你的故事，我覺得那種徹底停止是因為一種「強度的累積」，讓你在過程中迷失了初衷，喜悅感完全消失了。

Alisson Sandi: 沒錯，我就像台機器。我出身軍人家庭，當我決定要做某事時，我會切斷所有情感，只管執行。這很不健康。

Gui Prada: 巴西 Zouk 圈是社會的一個小縮影，我們在這裡遵循著社會的價值觀：像是「我應該達到某個境界」、「我應該變強」。這些目標導向的期待加上社群媒體的「比較心理」，會讓你覺得自己落後了。這會把你的核心初衷埋得越來越深。 就像疫情期間，大家有時間反思，很多人意識到：「我為什麼要做這個？」有些訓練極度刻苦的人在那之後就徹底停止了。他們發現自己只是在自動導航。

在我的故事中，我沒有徹底停止過，但我曾停止練習某些舞種。比如我曾沉迷探戈（Tango）好幾年，壓力大到讓我不想跳了，於是我休息了幾個月再回來。因為我的賭注沒有你那麼大。對你來說，舞蹈關乎簽證和生計；對我來說，舞蹈是幫助我面對生活的支柱。

另外一個現在讓我想聊聊的原因是「環境的變遷」。現在的 Zouk 圈跟 7、8 年前很不一樣。有些人適應不了這種變化，會覺得「我當初愛上的那個環境已經不在了」。比如我的第一位 Zouk 老師，她就很難接受現在的環境。這也可能成為人們離開的原因。

Alisson Sandi: 最危險的是讓環境控制你的決定。雖然我當時說我「沒得選」，但其實我有——我可以回巴西。但我選擇留下，並接受了隨之而來的後果。 自從我重新找回自己後，我現在專注於能讓我感到幸福的事。例如 Baza Zouk，這個計劃讓我充滿喜悅，我願意為它投入 24/7 的精力，因為它是不營利的，是回饋社群的。還有這個 Zouk Nerds Podcast，雖然工作量很大，但分享的過程讓我很快樂。

### 第三部分：\[[20:00](http://www.youtube.com/watch?v=FgKjYCFWX8g\&t=1200)] - \[[30:00](http://www.youtube.com/watch?v=FgKjYCFWX8g\&t=1800)]

Alisson Sandi: 社群中總會有人說「你應該做這個、應該做那個」，我現在必須不斷說「不」。 很多人說「我沒時間」，我都會解讀為「我不打算為此擠出時間」。因為對於真正想要的事，我們通常都能擠出時間。 在疫情中，很多人被拔掉了原本的生活插銷，意識到現在的職業或生活並不是他們想要的。學會「不去感受罪惡感」是非常難的一課。 兩年前有個朋友告訴我：「Alisson，你必須停止因為『優先考慮自己』而感到自責。」我以前常因為不想讓別人失望而繼續做某事，那並不快樂。 現在我會更理性、甚至冷靜地裁減計畫。如果裁掉它能改善我的睡眠、情緒和與家人的相處，我就會做。雖然這會讓某些人失望，但對我自己和我愛的人是有益的。

Gui Prada: 你還有跟那些以前很活躍、但現在完全消失的人聯絡嗎？

Alisson Sandi: 不常聯絡。這跟我的 ADHD 有關，我對於不在眼前或無法感知的事物很難保持連結。當我在澳洲，澳洲就是我的全部。很多 DC 的朋友會抱怨我不找他們，我很抱歉，但我當下的專注力就在那。

Gui Prada: 我很好奇他們在離開後的反思是什麼。

Alisson Sandi: 有的。他們分享了一種類似「鳥瞰」的視角。他們會懷念某些時刻，但離開的原因通常是更重大的：像是想要回歸家庭、生小孩、結婚。雖然懷念舞蹈，但生活重心的轉移（Priority Shift）更重要。 這對身為教學者的我們來說有點感傷，從商業角度看是流失了學生，但這就是生活。

Gui Prada: 我也想到那些「努力想要成為專業舞者卻失敗」的人。他們太想成功，這對心理健康造成了沈重負擔，導致他們現在只要想到舞蹈環境，就會想起那些阻礙。 我也認識一些疫情後停止教學、轉行做生意的人，兩年後（2024）他們又回來了，因為他們在離開的時間裡，重新感受到了舞蹈美好的一面。 所以「休息」真的很重要。有時候你必須站在外面看，才能看清你是否真的還喜歡它。

### 第四部分：\[[30:00](http://www.youtube.com/watch?v=FgKjYCFWX8g\&t=1800)] - 結尾 \[[45:05](http://www.youtube.com/watch?v=FgKjYCFWX8g\&t=2705)]

Alisson Sandi: 我想對處於領導地位的人（老師、主辦人）說幾句話。 當我還像台機器在運作時，如果有學生優先選擇了別的事，我會感到被背叛。我覺得：「我教了你這麼多年，你居然選擇別的？」 但現在我會說：「玩得開心！享受你的生活！」因為我知道這與我無關，這只是生活。 當我學會掌控自己的選擇後，我就不再覺得「為什麼別人可以休息而我不行」了。

Gui Prada: 當你全心全意想在一個地方，跟你不得不待在那裡，呈現出來的「存在感」（Presence）是完全不同的。如果你能誠實地面對內心，選擇減少舞蹈甚至離開，當你真的想回來時，那種狀態會更正面，會感染周圍的人。 很多老師會覺得「我在這個人身上浪費了這麼多時間，結果他不跳了」。但那並不是浪費，因為在發生的當下，你們都擁有過美好的經驗。如果你對未來有過高的期待，那期待就是「喜悅的賊」。

Alisson Sandi: 我現在嘗試專注於當下。雖然我有個充滿夢想的大腦，但我現在會準備 Plan A, B, C, D。 任何決定我都有多重方案和理由。即使事情不如預期，我也會確保自己在財務、情感和職業上是安全的。如果不安全，我甚至不會去冒險。

Gui Prada: 我也有 Plan A 到 G。我稱之為「多重宇宙」（Multiverse）。雖然有時候很難做決定，但這給了我安全感。 這些不同的路徑最終都會帶我到我想去的地方，只是有些長、有些短、有些獎勵較多。

Alisson Sandi: 你應該來我腦子待個五分鐘試試（笑）。

Gui Prada: 不了，謝謝。

Alisson Sandi: 好了，我想我們今天分享了很多深層且毫無保留的對話。希望能給正在經歷舞蹈或生活低潮的你一些指引。我們下一集見！

Gui Prada: 掰掰大家！
